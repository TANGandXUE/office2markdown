# 📄 MarkItDown 文档转换器

> 🎯 **Windows一键启动** | 基于Microsoft MarkItDown的智能文档转换工具

## ✨ 核心特性

- 🚀 **一键启动** - 双击即用，自动配置所有环境
- 🌐 **现代Web界面** - 基于Gradio的美观直观界面
- 📁 **批量处理** - 支持同时转换多个文档文件
- 🔄 **实时预览** - 即时查看转换结果和统计信息
- 🎯 **智能识别** - AI驱动的文件格式自动检测
- 💾 **便捷下载** - 单文件下载或批量ZIP打包

## 🚀 超简单使用

### 🎯 两种启动方式

#### 方式一：一键启动（推荐）
1. **下载项目** - 下载并解压到任意文件夹
2. **双击启动** - 双击 `test_start.bat` 文件
3. **开始转换** - 浏览器自动打开，拖拽文件即可转换

#### 方式二：手动启动
1. **打开命令行** - 在项目文件夹打开PowerShell/CMD
2. **切换目录** - `cd program`
3. **运行程序** - `python app.py`

> 💡 **首次使用** 会自动下载和配置所有依赖，可能需要几分钟
> 
> 🔄 **再次使用** 直接启动，几秒钟即可打开界面
> 
> ⚠️ **注意** - 如果 `test_start.bat` 无法运行，请尝试右键"以管理员身份运行"

## 🎯 使用指南

### 🔄 单文件转换
```
1. 点击 "单文件转换" 标签页
2. 拖拽或选择一个文档文件
3. 点击 "开始转换" 按钮
4. 查看转换结果统计信息
5. 复制内容或下载.md文件
```

### 📁 批量转换
```
1. 点击 "批量转换" 标签页  
2. 同时选择多个文档文件（支持拖拽）
3. 查看文件信息和支持状态
4. 点击 "批量转换" 按钮
5. 下载包含所有结果的ZIP文件
```

## 🐛 常见问题

### Q: 双击脚本没有反应？
- 确保文件名是 `test_start.bat`
- 右键选择"以管理员身份运行"
- 检查是否被杀毒软件拦截
- 如果仍有问题，请使用手动启动方式

### Q: 提示找不到Python？
- 脚本会提示下载链接：https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"
- 重启计算机后再次运行脚本

### Q: 转换失败怎么办？
- 检查文件是否损坏或受密码保护
- 查看错误信息确定具体原因
- 尝试将.doc文件另存为.docx格式

### Q: 界面无法打开？
- 检查地址是否为 http://localhost:7860
- 尝试关闭防火墙或杀毒软件
- 使用Chrome、Edge等现代浏览器
